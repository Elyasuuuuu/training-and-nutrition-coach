# training-and-nutrition-coach

OpenClaw skill for practical fitness coaching: workout plans, calorie and macro estimates, meal planning, and goal-aligned recipe generation.

## What it does

This skill helps an agent provide simple, actionable fitness and nutrition guidance.

It is designed for requests such as:

- workout plans
- fat loss guidance
- muscle gain guidance
- body recomposition support
- calorie and macro estimates
- meal planning
- recipe suggestions aligned with a physique goal

## Features

- beginner-friendly workout planning
- practical calorie and macro estimation
- realistic meal structure guidance
- recipe-oriented nutrition support
- reusable heuristics for common fitness goals
- compact response templates for clear output

## Included resources

- `SKILL.md` — core skill instructions
- `references/nutrition-baselines.md` — calorie and macro heuristics
- `references/output-templates.md` — output structure patterns
- `references/use-cases.md` — common real-world coaching cases
- `scripts/calc_macros.py` — simple helper for calorie/macro estimates

## Best for

- beginners
- users who want practical plans instead of theory
- simple and sustainable nutrition guidance
- users who want meals and recipes aligned with training goals

## Scope

This skill is intended for general fitness coaching support.

It does **not** replace:
- medical advice
- injury diagnosis
- treatment plans
- eating disorder care
- professional clinical nutrition care

## Example prompts

- Create a beginner workout plan for fat loss
- Estimate my calories and macros for body recomposition
- Build a simple meal plan for muscle gain
- Give me 3 high-protein recipe ideas for batch cooking
- Make me a weekly training and nutrition structure

## Philosophy

This skill favors:
- simplicity
- consistency
- sustainable habits
- practical recommendations
- useful action over fake precision
